## Getting started
Download the [latest release](https://github.com/hugolgst/digart/releases):

And then run the tool

```bash
$ ./digart
$ digart.exe
```

To change number you can just write the file path
```
resources/e.txt
resources/phi.txt
resources/pi.txt
```

```bash
$ ./digart resources/e.txt
```
